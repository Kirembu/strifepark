Let's explore the <canvas>

HTML5 <canvas> Checkers Game

Mark Pilgrim teaches us some basics of the <canvas> in the chapter "Let's Call It a Draw(ing Surface)" in his free online book - Dive Into HTML5. The chapter culminates with an awesome example - an addictive game called Halma. You can try Mark's Halma game here.

I decided to dissecet the Halma game and use the source code as the basis for my own Checkers game. On my blog, I'll be posting some discussions on how this code actually works, so stay tuned. Also, This game is kind of useless at the moment since both people have to sit at the same physical computer screen to play. I have plans to make this thing multiplayer as I learn more about HTML5. For now - Enjoy beating yourself in Checkers! Email me if you find any bugs.
